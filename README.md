## 使用方法

此脚本分为三个模式：
1. 加人模式：无视人数限制，直接加课（推荐这个方法）
2. 提醒模式：检测到有新的讲座的时候会自动发送邮件，批量信息发送给设置的用户（需要一直启动）
3. 蹲课模式：检测到有新的讲座的时候会自动抢课（需要一直启动）

### 加人模式

此模式打开lecture_select文件夹，按照`README`进行操作。
